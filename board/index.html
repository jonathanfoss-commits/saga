Lim inn hele eierprofil-dokumentet ditt her. Det lagres kun lokalt på denne enheten (aldri på noen server eller i koden) og blir styringsgrunnlaget for alle vurderinger AEIS gjør. SAGA-appen på samme enhet leser også profilen.
